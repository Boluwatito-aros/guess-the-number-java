# Guess the Number

A logic and reasoning game built in Java.

## About
Built a number-guessing game where the program generates a random secret number and
the player must identify it through guided hints.

## Features
- Binary-search-inspired hint system ('too high' / 'too low')
- Configurable difficulty — Easy (1–50), Medium (1–100), Hard (1–500)
- Attempts scale with difficulty level
- Tracks and displays number of attempts taken per round

## Tech
Java · OOP · Random number generation · Binary search logic
